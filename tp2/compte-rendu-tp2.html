Exercice 0 : Mon compte-rendu est trop stylé !

  1. Le titre de l'exercice courant se trouve normalement dans une balise balise h2. Ajoutez à cette balise
    un attribut style qui permet de mettre le titre de l'exercice en orange.

    J'ai ajouté l'attribut style="color: orange;" à la balise <h2> de l'exercice 5 pour mettre le titre en orange.

  2. Déclarez une balise style à l'intérieur des balises head du document, puis faites en sorte que tout le document utilise une police sans-serif.

    J'ai ajouté la balise <style> dans la section <head> du document avec la règle CSS suivante : body { font-family: Arial, sans-serif; } pour que tout le document utilise une police sans-serif.

  3. Faites en sorte que la balise h1 corresponde à une police de 24 points, la balise h2 à une police de 18 points
    et la balise h3 à une police 16 points.

    J'ai ajouté les règles CSS suivantes dans la balise <style> :
    h1 { font-size: 24pt; }
    h2 { font-size: 18pt; }
    h3 { font-size: 16pt; }
    pour définir les tailles de police respectives pour les balises h1, h2 et h3.

  4. Faites en sorte que les balises h2 soient affichées en bleu marine, à part le titre de cet exercice (qui doit rester en orange).

    J'ai ajouté la règle CSS suivante dans la balise <style> :
    h2 { color: navy; }
    pour que toutes les balises h2 soient affichées en bleu marine. Le titre de l'exercice 5 reste en orange grâce à l'attribut style spécifique ajouté précédemment.

  5. Faites en sorte que le titre du compte-rendu de TP soit centré.
    Pour cela, vous allez créer une classe "centered" et ajouter un attribut class à la balise englobant le titre du compte-rendu.
    Déclarez ensuite que le style pour la classe "centered" est centré.

    J'ai ajouté la classe "centered" à la balise <h1> du titre du compte-rendu et j'ai défini la règle CSS suivante dans la balise <style> :
    .centered { text-align: center; }
    pour centrer le titre du compte-rendu de TP.

  6. On veut que votre nom soit aligné à droite. Commencez par déclarer votre nom entre des balises div puis ajoutez une classe right permettant d'aligner le texte à droite.

    J'ai enveloppé mon nom dans une balise <div> avec la classe "right" et j'ai ajouté la règle CSS suivante dans la balise <style> :
    .right { text-align: right; }
    pour aligner mon nom à droite.

  7. Faites en sorte que le lien vers le cours de programmation web (que vous avez créé à l'exercice 1) soit en rouge si la souris
    n'est pas sur le lien et en bleu canard si la souris survole le lien. Conseil : Vous utiliserez pour cela les pseudo-classes des liens.

    J'ai ajouté les règles CSS suivantes dans la balise <style> :
    a:link { color: red; }
    a:hover { color: teal; }
    pour que le lien vers le cours de programmation web soit en rouge lorsqu'il n'est pas survolé par la souris et en bleu canard lorsqu'il est survolé.

Exercice 1 : Feuille de style externe, liste et table

  1. Reprendre le compte-rendu HTML créé au TP précédent, pour n'en conserver que l'en-tête et le dernier exercice,
    que vous renommerez Exercice 0 : Mon compte-rendu est trop stylé ! et dont vous supprimerez les informations de style.

    J'ai repris le compte-rendu HTML du TP précédent en ne conservant que l'en-tête et le dernier exercice,
    que j'ai renommé Exercice 0 : Mon compte-rendu est trop stylé !. J'ai également supprimé les informations de style de ce compte-rendu.

  2. On veut maintenant créer une feuille de style externe, c'est-à-dire une feuille de style déclarée non pas à l'intérieur
    de compte-rendu-tp2.html mais dans un fichier externe nommé compte-rendu-tp2.css.
    Modifiez votre travail pour que le compte-rendu utilise ce fichier de style.

    J'ai créé un fichier externe nommé compte-rendu-tp2.css et j'ai déplacé toutes les règles CSS de la balise <style> du document HTML vers ce fichier.
    Ensuite, j'ai ajouté la ligne suivante dans la section <head> du document HTML pour lier la feuille de style externe :
    <link rel="stylesheet" href="compte-rendu-tp2.css">.

  3. Faites en sorte que la liste des questions de cet exercice et de l'exercice précédent utilise les chiffres romains
    (en minuscules) plutôt que les traditionnels chiffres 1, 2, 3, etc.

    J'ai ajouté la règle CSS suivante dans le fichier compte-rendu-tp2.css :
    ol { list-style-type: lower-roman; }
    pour que les listes ordonnées utilisent des chiffres romains en minuscules.

  4. Le site https://en.wikipedia.org/wiki/List_of_2021_box_office_number-one_films_in_the_United_States contient
    un tableau des films au box office. On souhaite recopier le titre et le montant des recettes pour les 5 plus gros succès
    (il y a des flèches pour trier) pour les mettre dans une table dont nous allons changer l'aspect.

    Titre du film Montant des recettes
    Spider-Man: No Way Home 804 779 335 $
    Shang-Chi and the Legend of the Ten Rings 224 543 292 $
    Venom : Let There Be Carnage 212 638 437 $
    Black Widow 183 651 655 $
    Fast and Furious 9 173 005 945 $

    J'ai copié le titre et le montant des recettes pour les 5 plus gros succès du tableau sur la page Wikipedia et je les ai organisés dans une table HTML.

  5. Faites en sorte que la table occupe toute la largeur du compte rendu.

    J'ai ajouté la règle CSS suivante dans le fichier compte-rendu-tp2.css :
    table { width: 100%; }
    pour que la table occupe toute la largeur du compte-rendu.

  6. Centrez les titres et changer la police de caractères des titres pour utiliser des petites majuscules (small caps).

    J'ai ajouté les règles CSS suivantes dans le fichier compte-rendu-tp2.css :
    th { text-align: center; font-variant: small-caps; }
    pour centrer les titres de la table et utiliser des petites majuscules pour les titres.

  7. Aérez la table avec la propriété padding pour que le texte ne soit pas collé aux bordures des cases
    (qui sont invisibles a priori, mais que vous pouvez toujours rendre visibles si ça vous aide).

    J'ai ajouté la règle CSS suivante dans le fichier compte-rendu-tp2.css :
    td { padding: 10px; }
    pour ajouter de l'espace entre le texte et les bordures des cases de la table.

  8. Faites en sorte que les valeurs du tableau soient alignées à gauche, sauf celles de la dernière colonne, qui doivent être centrées.

    J'ai ajouté les règles CSS suivantes dans le fichier compte-rendu-tp2.css :
    td:first-child { text-align: left; }
    td:last-child { text-align: center; }
    pour aligner les valeurs de la première colonne à gauche et celles de la dernière colonne au centre.

  9. Changez la couleur de fond des lignes paires (even) pour mettre du bleu (80% de rouge, 80% de vert et 90% de bleu) en utilisant la pseudo-classe :nth-child.

    J'ai ajouté la règle CSS suivante dans le fichier compte-rendu-tp2.css :
    tr:nth-child(even){ background-color: rgb(80%, 80%, 90%); }
    pour changer la couleur de fond des lignes paires de la table.

  10. Faites en sorte que si l'on survole une ligne avec la souris la couleur du fond corresponde à 28 sur 255 de rouge, 120 sur 255 de vert et 12 sur 255 de bleu en utilisant un codage hexadécimal.

    J'ai ajouté la règle CSS suivante dans le fichier compte-rendu-tp2.css :
    tr:hover { background-color: #1C780C; }
    pour changer la couleur de fond d'une ligne lorsqu'on la survole avec la souris.

Exercice 2 : La ferme des animaux

  1. Récupérez six images d'animaux dans un fichier zippé ainsi que le fichier exo2.html.
    Vérifiez que les images s'affichent correctement lorsque vous visualisez le document.
    Note 1 : On rappelle que le fichier exo2.html ne doit être modifié que en ajoutant des classes aux boîtes,
    et des informations de style, la structure du document doit rester la même.
    Note 2 : Le fichier resultat-exo2.png correspond à l'affichage que l'on souhaite avoir à la fin de cet exercice.

    J'ai récupéré les six images d'animaux et le fichier exo2.html, puis j'ai vérifié que les images s'affichent correctement en visualisant le document dans un navigateur.

  2. Faites en sorte que toutes les images à l'intérieur des boîtes (pas les autres) aient une largeur (width) de 180 pixels.
    Note : on suppose que l'on peut ajouter des images pas dans les boîtes par la suite.

    J'ai ajouté la règle CSS suivante dans la balise <style> du fichier exo2.html :
    .animal-box img { width: 180px; }
    pour définir la largeur de toutes les images à l'intérieur des boîtes à 180 pixels, sans affecter les autres images qui pourraient être ajoutées ultérieurement.

  3. Au lieu d'afficher les images verticalement, on souhaite les afficher horizontalement. Utilisez pour cela l'attribut float

    J'ai ajouté la règle CSS suivante dans la balise <style> du fichier exo2.html :
    .animal-box { float: left; }
    pour faire en sorte que les images à l'intérieur des boîtes s'affichent horizontalement.

  4. Que se passe-t-il si l'on fait varier la taille de la fenêtre du navigateur horizontalement ?

    Lorsque la taille de la fenêtre du navigateur est réduite horizontalement, les images flottantes s'adaptent en se réorganisant pour s'ajuster à l'espace disponible.
    Si la fenêtre devient trop étroite, les images peuvent se déplacer vers la ligne suivante, créant ainsi un effet de "wrap" où les images s'empilent verticalement au lieu de rester sur une seule ligne.

  5. Vérifiez que vous pouvez aussi mettre toutes les images à droite en changeant juste un mot dans la partie CSS du document.
    Puis remettez les images alignées à gauche

    J'ai changé la règle CSS suivante dans la balise <style> du fichier exo2.html :
    .animal-box { float: right; }
    pour aligner les images à droite, puis j'ai remis la règle à float: left; pour les aligner à gauche à nouveau.

  6. Faites en sorte que les titres (animal1, animal2, etc.) soient affichés en blanc sur fond noir et soient centrés horizontalement par rapport
    à la boîte qui les contient. Jouez de plus avec la propriété padding pour avoir un espacement convenable.

    J'ai ajouté les règles CSS suivantes dans la balise <style> du fichier exo2.html :
    .animal-box div { color: white; background-color: black; text-align: center; padding: 5px; }
    pour que les titres soient affichés en blanc sur fond noir, centrés horizontalement et avec un espacement de 5 pixels grâce à la propriété padding.

  7. Espacez les unes des autres les six boîtes contenant les images avec la propriété margin.
    Attention : on ne veut pas ajouter de marge aux six boîtes ne contenant pas d'image, par exemple les six boîtes contenantles titres des images.

    J'ai ajouté la règle CSS suivante dans la balise <style> du fichier exo2.html :
    .animal-box { margin: 10px; }
    pour ajouter une marge de 10 pixels autour de chaque boîte contenant les images, sans affecter les boîtes contenant les titres.

  8. Faites en sorte que les images à l'intérieur des boîtes aient une bordure de trois pixels de largeur de couleur #aaaaaa.

    J'ai ajouté la règle CSS suivante dans la balise <style> du fichier exo2.html :
    .animal-box img { border: 3px solid #aaaaaa; }
    pour ajouter une bordure de trois pixels de largeur de couleur #aaaaaa à toutes les images à l'intérieur des boîtes.

  9. Changez l'opacité (propriété opacity) lorsque l'utilisateur passe sa souris au-dessus d'une image dans une boîte.

    J'ai ajouté la règle CSS suivante dans la balise <style> du fichier exo2.html :
    .animal-box img:hover { opacity: 0.5; }
    pour changer l'opacité des images à 0.5 lorsque l'utilisateur passe sa souris au-dessus d'une image dans une boîte.

  10. Faites en sorte que les titres soient accolés par le bas à l'image et mordent légèrement sur l'image (de 5 pixels par exemple).
    Note: l'idée est de placer la boîte contenant le texte relativement par rapport à la boîte contenant l'image.

    J'ai ajouté la règle CSS suivante dans la balise <style> du fichier exo2.html :
    .animal-box div { position: relative; top: -5px; }
    pour faire en sorte que les titres soient accolés par le bas à l'image et mordent légèrement sur l'image de 5 pixels.

  11. Faites en sorte que le texte à mettre après soit situé en-dessous des images et de leurs titres, et non pas à leur droite.

    J'ai ajouté la règle CSS suivante dans la balise <style> du fichier exo2.html :
    .end { clear: both; }
    pour faire en sorte que le texte à mettre après soit situé en-dessous des images et de leurs titres, en utilisant la propriété clear pour éviter que le texte ne s'affiche à côté des éléments flottants.

Exercice 3 : Barre de menu

  1. Faites en sorte que la liste ait une position fixe, en haut à gauche de la partie affichage du navigateur.

    J'ai ajouté la règle CSS suivante dans la balise <style> du fichier exo3.html :
    ul { position: fixed; top: 0; left: 0; }
    pour faire en sorte que la liste ait une position fixe en haut à gauche de la partie affichage du navigateur.

  2. Retirez le style par défaut des items de liste (avec list-style-type) pour qu'il n'y ait pas de style par défaut.

    J'ai ajouté la règle CSS suivante dans la balise <style> du fichier exo3.html :
    ul { list-style-type: none; }
    pour retirer le style par défaut des items de liste.

  3. Affichez les items en gras et avec une police Arial (ou sinon sans-serif).

    J'ai ajouté la règle CSS suivante dans la balise <style> du fichier exo3.html :
    ul li { font-weight: bold; font-family: Arial, sans-serif; }
    pour afficher les items en gras et avec une police Arial (ou sinon sans-serif).

  4. Faites en sorte que les items soient affichés horizontalement collés à gauche et non plus verticalement.

    J'ai ajouté la règle CSS suivante dans la balise <style> du fichier exo3.html :
    ul li { float: left; }
    pour faire en sorte que les items soient affichés horizontalement collés à gauche et non plus verticalement.

  5. Faites en sorte que les items aient une largeur de 100 pixels et ajoutez un espacement d'une dizaine de pixel entre les items.

    J'ai ajouté la règle CSS suivante dans la balise <style> du fichier exo3.html :
    ul li { width: 100px; margin-right: 10px; }
    pour faire en sorte que les items aient une largeur de 100 pixels et ajouter un espacement d'une dizaine de pixel entre les items.

  6. Changez le fond du menu pour qu'il soit de couleur violette (purple), en chantant purple rain, et changez la couleur des items pour qu'ils soient blancs.

    J'ai ajouté la règle CSS suivante dans la balise <style> du fichier exo3.html :
    ul { background-color: purple; }
    ul li { color: white; }
    pour changer le fond du menu en violette et la couleur des items en blanc.

  7. Fixez la hauteur (height) du menu à 50 pixels puis faites en sorte que le menu occupe toute la largeur (width).

    J'ai ajouté la règle CSS suivante dans la balise <style> du fichier exo3.html :
    ul { height: 50px; width: 100%; }
    pour fixer la hauteur du menu à 50 pixels et faire en sorte qu'il occupe toute la largeur.

  8. Faites en sorte que l'item 'About' soit affiché à droite de la barre de menu.

    J'ai ajouté la règle CSS suivante dans la balise <style> du fichier exo3.html :
    ul li:last-child { float: right; }
    pour faire en sorte que l'item 'About' soit affiché à droite de la barre de menu.

  9. Affichez une ligne de 2 pixels en bas du menu.
    L'astuce est de mettre une bordure de 2 pixels seulement en bas.

    J'ai ajouté la règle CSS suivante dans la balise <style> du fichier exo3.html :
    ul { border-bottom: 3px solid black; }
    pour afficher une ligne de 2 pixels en bas du menu en utilisant une bordure uniquement en bas.

Conclusion

Ce TP m'a permis de me familiariser avec les bases du CSS, notamment la création de styles pour différents éléments HTML, l'utilisation de classes et d'identifiants,
ainsi que la manipulation de la mise en page avec des propriétés telles que float et position. J'ai également appris à créer une feuille de style externe pour séparer le contenu HTML de la présentation CSS,
ce qui rend le code plus organisé et maintenable. Enfin, j'ai découvert comment utiliser les pseudo-classes pour ajouter des effets interactifs aux éléments de la page, comme changer l'apparence d'un lien au survol de la souris.
