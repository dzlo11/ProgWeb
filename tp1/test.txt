              <p> J'ai ajouté la balise &lt;style&gt; dans la section &lt;head&gt; du document avec la règle CSS suivante : body { font-family: Arial, sans-serif; } pour que tout le document utilise une police sans-serif.</p>
          </li>
          <li>
            <h3>Faites en sorte que la balise h1 corresponde à une police de 24 points, la balise h2 à une police de 18 points <br>et la balise h3 à une police 16 points.</h3>
              <p> J'ai ajouté les règles CSS suivantes dans la balise &lt;style&gt; : <br>
                h1 { font-size: 24pt; } <br>
                h2 { font-size: 18pt; } <br>
                h3 { font-size: 16pt; } <br>
                pour définir les tailles de police respectives pour les balises h1, h2 et h3.</p>
          </li>
          <li>
            <h3>Faites en sorte que les balises h2 soient affichées en bleu marine, à part le titre de cet exercice (qui doit rester en orange).</h3>
              <p> J'ai ajouté la règle CSS suivante dans la balise &lt;style&gt; : <br>
                h2 { color: navy; } <br>
                pour que toutes les balises h2 soient affichées en bleu marine. Le titre de l'exercice 5 reste en orange grâce à l'attribut style spécifique ajouté précédemment.</p>
          </li>
          <li>
            <h3>Faites en sorte que le titre du compte-rendu de TP soit centré. <br>
              Pour cela, vous allez créer une classe "centered" et ajouter un attribut class à la balise englobant le titre du compte-rendu. <br> Déclarez ensuite que le style pour la classe "centered" est centré.</h3>
              <p> J'ai ajouté la classe "centered" à la balise &lt;h1&gt; du titre du compte-rendu et j'ai défini la règle CSS suivante dans la balise &lt;style&gt; : <br>
                .centered { text-align: center; } <br>
                pour centrer le titre du compte-rendu de TP.</p>
          </li>
