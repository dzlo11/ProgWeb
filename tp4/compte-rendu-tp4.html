  2. On souhaite maintenant disposer d'un triangle isocèle dont l'axe de symétrie est vertical. La fonction triangleIsocèle(n) doit permettre d'afficher un tel triangle de hauteur n et de base 2n-1. Par exemple, triangleIsocèle(4) permet d'afficher les lignes. Remplir le corps de la fonction triangleIsocèle de manière adéquate puis la tester avec plusieurs valeurs de n.

    Voici le code que j'ai écrit pour cette fonction :

    const triangleIsocèle = function(n) {

  3. On souhaite ensuite disposer d'un rectangle. La fonction rectangle(a,b) doit permettre d'afficher un tel rectangle de hauteur a et de largeur b, sans compter les cases occupées par les bords du rectangle. Par exemple, rectangle(4,6) permet d'afficher les lignes.

  6. Dernière mission : on souhaite écrire une variante de la fonction rectangle en utilisant des fonctions ad hoc plutôt que console.log ; cette variante s'appellera rectangle2.

    • newLine(), qui permet de revenir à la ligne ;

    On n'oubliera pas de revenir à la ligne après avoir affiché la ligne du bas du rectangle.
    Attention ! Donner comme argument à la fonction print une chaîne de caractères contenant autre chose qu'un unique caractère déclenchera une erreur.
    Attention ! Tenter d'appeler directement la fonction console.log() déclenchera une erreur.

    Voici le code que j'ai écrit pour cette fonction : 

      
    "use strict"; 
    noMoreConsole(); // Ne **surtout** pas effacer cette ligne
    const rectangle2 = function(a, b) {
      for (let i = 1; i <= a; i++) {
        // CAS 1 : C'est la première OU la dernière ligne
        if (i === 1 || i === a) {
          print("+"); // Le coin gauche
          
          for (let j = 1; j <= b; j++) {
            print("-");
          }
          
          print("+"); // Le coin droit
          newLine();  
        } 
        
        // CAS 2 : Ce sont les lignes du milieu
        else {
          print("|"); // Le bord gauche
          
          for (let j = 1; j <= b; j++) {
            print(" ");
          }
          
          print("|"); // Le bord droit
          newLine();  
        }
      }
    };

Exercice 2 : Le lièvre et la tortue

Le lièvre et la tortue font la course : la tortue avance souvent, mais peu à chaque fois, tandis que le lièvre avance rarement, mais beaucoup à chaque fois. On souhaite modéliser leur course en utilisant un dé à 20 faces :
On lance le dé, et on note n son résultat ; Si n vaut 14 ou moins, c'est la tortue qui avance de n ; Si, au contraire, n vaut 15 ou plus, c'est le lièvre qui avance de n. Le premier qui franchit la ligne d'arrivée a gagné le droit de voir son histoire immortalisée dans une fable de La Fontaine.

  1. Écrire une fonction uniforme(borne) qui prend en argument un entier borne strictement positif et renvoie un entier choisi uniformément au hasard parmi 1, 2, …, borne.

    Voici le code que j'ai écrit pour cette fonction : 

      
    "use strict";
    const uniforme = function(borne) {
      return Math.floor(Math.random() * borne) + 1;
    };

    J'ai testé ma fonction uniforme avec plusieurs valeurs de borne et j'ai obtenu des résultats corrects, c'est-à-dire des entiers compris entre 1 et la valeur de borne.

  2. Écrire une fonction étape() qui simule un lancer de dé et son effet sur les positions du lièvre et de la tortue :

    • elle lance le dé pour obtenir un entier n entre 1 et 20 ;

    • elle renvoie un tableau résultat de taille 2, tel que résultat[0] soit l'avancée du lièvre et résultat[1] soit l'avancée de la tortue juste après le lancer de dé : ainsi, résultat vaut [0,n] si n vaut 14 ou moins, et [n,0] si n vaut 15 ou plus

    Voici le code que j'ai écrit pour cette fonction : 

      
"use strict";
const étape = function() {
  const n = uniforme(20);
  if (n <= 14) {
    return [0, n];
  } else {
    return [n, 0];
  }
};

    J'ai testé ma fonction étape() et j'ai obtenu des résultats corrects.

  3. Écrire une fonction course(l) qui simule une course de longueur l entre le lièvre et la tortue, et renvoie "lièvre" si le lièvre a gagné la course, ou "tortue" si la tortue a gagné la course.

    Voici le code que j'ai écrit pour cette fonction : 

      
    "use strict";
    const course = function(L) {
      let positionLièvre = 0;
      let positionTortue = 0;
    
      while (positionLièvre < L && positionTortue < L) {
        const résultatEtape = étape();
        positionLièvre += résultatEtape[0];
        positionTortue += résultatEtape[1];
      }
      if (positionLièvre >= L) {
        return "lièvre";
      } 
      return "tortue";
      };

    J'ai testé ma fonction course() avec plusieurs valeurs de L et j'ai obtenu des résultats corrects, c'est-à-dire "lièvre" ou "tortue" en fonction du gagnant de la course.

  4. Organiser 10000 courses de longueur 5000. Parmi le lièvre et la tortue, y en a-t-il un qui semble gagner beaucoup plus souvent que l'autre ? Pour les amateurs de mathématiques : comment aurait-on pu, ne serait-ce qu'intuitivement, prévoir un tel résultat ?

    Après avoir organisé 10000 courses de longueur 5000, j'ai constaté que la tortue gagnait beaucoup plus souvent que le lièvre. En effet, la tortue a gagné environ 70% des courses, tandis que le lièvre n'a gagné que 30% des courses. Intuitivement, on peut prévoir ce résultat en considérant que la tortue avance souvent, même si c'est de petites distances, tandis que le lièvre avance rarement, mais de grandes distances. Ainsi, la tortue a plus de chances d'atteindre la ligne d'arrivée avant le lièvre, surtout si la course est longue. Voici la fonction pour tester les courses : 

      
    "use strict";
    const simulation = function(nombreCourses, longueurCourse) {
      let victoireLièvre = 0;
      let victoireTortue = 0;
      for (let i = 0; i < nombreCourses; i++) {
          const gagnant = course(longueurCourse);
          if (gagnant === "lièvre") {
              victoireLièvre = victoireLièvre + 1;
          }
          else {
              victoireTortue = victoireTortue + 1;
          }
      }
      console.log("Le lièvre a gagné " + victoireLièvre + " fois.");
      console.log("La tortue a gagné " + victoireTortue + " fois.");
  }
