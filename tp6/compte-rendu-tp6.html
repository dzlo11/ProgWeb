Pour l'instant, nous allons utiliser la fonction shuffleCards qui vous a été donnée car cela sera plus facile pour la mise au point du programme. Que fait la fonction shuffleCards, et en quoi cela aide-t-il pour la mise au point du programme ?

    La fonction shuffleCards génère un tableau contenant deux fois chaque index d'image (0, 0, 1, 1, 2, 2, etc.). Pour l'instant, elle place les paires côte à côte sans les mélanger. Cela aide énormément pour la mise au point car nous connaissons à l'avance l'emplacement des paires. Si nous cliquons sur la carte 0 et la carte 1, nous savons qu'elles doivent correspondre. 

    const shuffleCards = function(length) {
  let cards = [];
  for(let i = 0; i < length; i++) {
    cards.push(i); // Ajoute le premier exemplaire de l'image
    cards.push(i); // Ajoute le deuxième exemplaire pour créer la paire
  }
  return cards;
};

  8. Enfin, modifiez shuffleCards pour que le tableau des cartes soit mélangé de façon aléatoire (comme dans la question 1 de l'exercice 3 du TP4).

    Voici les modification apportées à la fonction shuffleCards : 

    const shuffleCards = function(length) {
  let cards = [];

  for(let i = 0; i < length; i++) {
    cards.push(i);
    cards.push(i);
  } 

  // on réalise le tri aléatoire 
  for (let i = cards.length - 1; i > 0; i--) {
    let j = Math.floor(Math.random() * (i + 1)); // On choisit un index au hasard entre 0 et i
    
    // On échange l'élément à la position i avec celui à la position j
    let temporaire = cards[i];
    cards[i] = cards[j];
    cards[j] = temporaire;
  }

  return cards; 
};
