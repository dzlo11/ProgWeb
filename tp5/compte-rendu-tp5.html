Exercice 3 : MinMax, génération aléatoire et calcul rapide

  1. On cherche à écrire une fonction minmax qui prend en paramètre un tableau, calcule le minimum et le maximum de ce tableau et renvoie les deux valeurs sous forme d'un nouveau tableau à 2 cases dont la première case est le minimum et la seconde case est le maximum.
    Pour des questions d'efficacité, on veut que le calcul du minimum et du maximum soit fait lors du même parcours du tableau.
    Écrivez, dans un premier temps, l'algorithme que vous allez implanter en pseudo-code, puis écrivez le code JavaScript correspondant.
    Note : il existe les fonctions Math.min et Math.max.

    Voici la fonction minmax :

    "use strict";
  const minmax = fucntion(array) {
    if (array.length === 0) {
      return [undefined, undefined];
    }
    let min = array[0];
    let max = array[0];
    for (let i = 1; array.length; i++) {
      if (array[i] < min) {
        min = array[i];
      }
      if (array[i] > max) {
        max = array[i];
      }
    }
    return [min, max];
  };

    La fonciton minmax est une fonction assez efficace en temps, car elle réalise le calcul du minimum et du maximum en un seul parcours de tableau.

  2. Au lieu d'écrire la boucle à la main, on peut vouloir directement utiliser l'opérateur spread (…) pour calculer le minimum et le maximum du tableu.

  3. On souhaite pouvoir mélanger les cases d'un tableau dans le but de tester plus de combinaisons. L'idée de l'algorithme de mélange est la suivante : pour chaque case i, on tire un nombre aléatoire entre i et n - 1 (n étant le nombre de cases du tableau) et l'on va permuter (échanger si vous préférez) la case courante avec la case à l'index du nombre que l'on vient de tirer aléatoirement.
    Si l'on fait cela pour toutes les cases, le tableau sera mélangé (et on peut même s'offrir la joie de démontrer que ce tableau est bien échantillonné uniformément au hasard parmi tous les tableaux possibles).
    Écrivez dans un premier temps une fonction swap qui permet d'échanger deux cases du tableau.
    Puis écrivez la fonction randomizeArray qui mélange les valeurs du tableau pris en paramètre.
    Note : tirer un nombre aléatoire entre 0 (inclus) et n (exclu) en JavaScript se fait avec la fonction suivante :

    Voici la fonction swap : 

    "use strict";
  const swap = function(array, i, j) {
    let temp = array[i];
    array[i] = array[j];
    array[j] = temp;
  };

    Voici la fonction randomizeArray :

    "use strict";
  const random = n => Math.floor(n * Math.random());
  const randomizeArray = function(array) {
    for (let i = 0; i < array.length; i++) {
      const j = i + random(n - i);
      swap(array, i, j);
    }
  };

    La fonction randomizeArray parcourt le tableau. Pour chaque position i, on choisit un indice aléatoire entre i et la fin du tableau. Puis on échange les deux cases grâce à la fonction swap. Cela permet de mélanger correctement le tableau.

  4. Il est possible de diminuer le nombre de tests faits par la fonction minmax. En effet, si au lieu de faire évoluer le min et le max pour chaque case du tableau, on commence par lire deux valeurs de cases consécutives, regarder quelle est la plus grande puis la comparer avec le max, dans ce cas, la valeur la plus petite ne doit être comparée qu'avec le min donc chaque case n'a pas besoin d'être comparée et avec le max et avec le min.
    Écrivez dans un premier temps l'algorithme en pseudo-code.
    Note : pour éviter de dupliquer le code, on peut écrire le code si une valeur est plus grande que l'autre et dans l'autre cas, permuter les valeurs (attention, les valeurs, pas les cases du tableau) pour se retrouver dans le même cas.
    Vérifiez que votre algo fonctionne que le nombre de cases soit pair ou impair. Enfin, écrivez la fonction minmax3.

    Voici pseudo-code de l'agorithme : 

    Si le tableau est vide
  retourner [undefined, undefined]

  Initialiser min et max avec la première valeur du tableau

  Pour chaque élément du tableau à partir du second
    si l'élément est plus petit que min
      mettre à jour min
    si l'élément est plus grand que max
      mettre à jour max

  Retourner [min, max]

    Voici la fonction minmax3 :

    "use strict";
const minmax3 = function(array) {
  if (array.length === 0) {
    return [undefined, undefined];
  }
  
  let min, max, indexDepart;
  
  // Si le nombre d'éléments est impair
  if (array.length % 2 !== 0) {
    min = array[0];
    max = array[0];
    indexDepart = 1;
  } else {
    // Si pair, on compare les deux premiers pour initialiser
    if (array[0] < array[1]) {
      min = array[0]; max = array[1];
    } else {
      min = array[1]; max = array[0];
    }
    indexDepart = 2;
  }

  // On parcourt le reste du tableau deux par deux
  for (let i = indexDepart; i < array.length; i += 2) {
    let a = array[i];
    let b = array[i + 1];
    
    if (a < b) {
      if (a < min) min = a;
      if (b > max) max = b;
    } else {
      if (b < min) min = b;
      if (a > max) max = a;
    }
  }
  return [min, max];
};
